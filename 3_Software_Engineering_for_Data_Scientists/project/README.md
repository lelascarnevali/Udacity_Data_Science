# Project: Data Science Dashboard

## Overview

Build a dashboard web application that allows managers to monitor an employee's performance and their predicted risk of recruitment.

## Requirements

- Load and process employee performance data
- Train a machine learning model to predict recruitment risk
- Build an interactive web dashboard using FastHTML
- Display individual employee performance metrics and risk scores
- Allow managers to explore and filter employee data

## Deliverables

- [ ] Machine learning model for recruitment risk prediction
- [ ] FastHTML web application with interactive dashboard
- [ ] Deployed (or locally runnable) dashboard

## Getting Started

Add project files to this folder and work from the provided starter code.
