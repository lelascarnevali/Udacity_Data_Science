# Project: Data Science Pipeline

## Overview

Create a machine learning model pipeline with scikit-learn, using numeric and text data to predict whether or not a customer would recommend a product.

## Requirements

- Build a scikit-learn pipeline combining numeric and text feature processing
- Handle preprocessing, feature engineering, and model training in a single pipeline
- Prevent data leakage with proper pipeline design
- Evaluate the pipeline with appropriate metrics
- Ensure reproducibility

## Deliverables

- [ ] Scikit-learn pipeline combining numeric and text features
- [ ] Model evaluation report
- [ ] Reproducible notebook or script
