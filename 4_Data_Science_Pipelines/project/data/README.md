# Data

This project uses the review dataset provided with the Udacity Data Science
Pipeline project.

## Files

- `reviews.csv`: anonymized women's clothing reviews used to predict
  `Recommended IND`.

## Columns

- `Clothing ID`
- `Age`
- `Title`
- `Review Text`
- `Positive Feedback Count`
- `Division Name`
- `Department Name`
- `Class Name`
- `Recommended IND`

## Folder Policy

Treat `reviews.csv` as source data for this project. Do not edit the CSV by
hand; update transformations in the notebook pipeline instead.
