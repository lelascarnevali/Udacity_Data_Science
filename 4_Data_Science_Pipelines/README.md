# Course 4: Data Science Pipelines

Streamline complex machine learning projects by designing pipelines that combine preprocessing, modeling, and evaluation. Learn to manage numeric, image, and text data efficiently while preventing data leakage and boosting reproducibility.

**Estimated time:** ~18h

## Lessons

| # | Lesson | Description |
|---|--------|-------------|
| 1 | Scikit-Learn Pipelines | Efficient data workflows, preprocessing, feature engineering, and model optimization |
| 2 | Computer Vision Pipelines | Image preprocessing, feature extraction, and classification pipelines with SVMs and CNNs using OpenCV and PyTorch |
| 3 | NLP Pipelines | ML pipelines with text data: tokenization, vectorization, and part-of-speech tagging with spaCy |

## Project

**Project: Data Science Pipeline**

Create a machine learning model pipeline with scikit-learn, using numeric and text data to predict whether or not a customer would recommend a product.

See [`project/README.md`](project/README.md) for details.
