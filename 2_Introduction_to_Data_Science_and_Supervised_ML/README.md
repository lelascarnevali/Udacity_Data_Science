# Course 2: Introduction to Data Science and Supervised Machine Learning

Learn core data science concepts and apply supervised machine learning models in real projects. Build your portfolio, master evaluation techniques, and gain practical skills to stand out in data-driven careers.

**Estimated time:** ~10h 37min

## Lessons

| # | Lesson | Description |
|---|--------|-------------|
| 1 | The Data Science Process | CRISP-DM process and how to apply it to many data science problems |
| 2 | Supervised Machine Learning Algorithms | Regression, classification, linear models, decision trees, random forests, and neural networks with scikit-learn |
| 3 | Machine Learning Model Evaluation | Alternative metrics beyond default accuracy for real-world datasets |
| 4 | Model Interpretability and Fairness | Feature importances, SHAP values, and the Aequitas framework |
| 5 | Communicating to Stakeholders | GitHub repository and Medium blog post to communicate findings |

## Project

**Project: Data Science Blog Post - CMS Hospital Quality Analysis**

Complete the CRISP-DM process with a dataset of your choice, and deploy your findings in the format of a blog post.
