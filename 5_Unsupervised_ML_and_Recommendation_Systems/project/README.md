# Project: Recommendation System

## Overview

Implement a system for recommending technical articles from the IBM Watson platform, using ranking-based, content-based, and collaborative filtering approaches.

## Approaches

1. **Rank-Based Recommendations** — Recommend the most popular articles to new users.
2. **User-User Collaborative Filtering** — Find users similar to a given user and recommend articles they interacted with.
3. **Content-Based Filtering** — Use NLP to recommend articles similar to ones the user has already read.
4. **Matrix Factorization (SVD)** — Decompose the user-article interaction matrix to uncover latent features.

## Deliverables

- [ ] Jupyter Notebook implementing all four recommendation approaches
- [ ] Evaluation of each approach (online and/or offline)
- [ ] Analysis of trade-offs between methods

## Getting Started

Add the IBM Watson articles dataset to this folder and work from the notebook.
